# Time Series Analysis & Forecasting

Bike Rental Demand Forecasting with RNN, LSTM & GRU

This project explores deep learning approaches for time series forecasting using Recurrent Neural Network (RNN), Long Short-Term Memory (LSTM) and Gated Recurrent Unit (GRU) networks to predict bike rental demand based on historical usage and environmental conditions.

**Project Overview**

The objective of this project is to:
- Forecast hourly/daily bike rental demand
- Compare the performance of RNN LSTM and GRU architectures, along with their modified version (hidden layer modifications and hyperparameters tuning)
- Analyze temporal patterns in rental behavior


**Tech stack**
Python
TensorFlow / Keras
Pandas & NumPy
Scikit-learn
Matplotlib
